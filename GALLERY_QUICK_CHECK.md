# 🎯 GALLERY NAVIGATION - QUICK REFERENCE CARD

## 🔥 WHAT YOU MUST SEE (3 Things!)

### 1. PULSING WHITE BUTTONS ⭐ MOST IMPORTANT
When modal opens, look for:
- **Large white circles** on LEFT and RIGHT
- **Pulsing animation** (gentle pulse)
- **White borders** around buttons

**If you don't see these = SOMETHING IS WRONG!**

### 2. KEYBOARD HINT
At bottom center:
- **"Use ← → arrow keys to navigate"**
- Black background, pulsing

### 3. POSITION COUNTER
In modal header:
- **"5 / 30"** format
- Updates when navigating

---

## 🎮 How to Use

### Mouse
- Click **LEFT button** (←) = Previous
- Click **RIGHT button** (→) = Next

### Keyboard
- Press **←** = Previous
- Press **→** = Next
- Press **Esc** = Close

### Search
- Type: "Google", "AWS", "DAP", "DICT"
- Finds by organization

---

## 🐛 If Not Working

### Quick Fix
1. Press `F12` (open DevTools)
2. Click Console tab
3. Click any image
4. Look for: `🎹 Keyboard navigation enabled`
5. Press arrow keys
6. Look for: `🔑 Key pressed: ArrowLeft`

**See these messages?** = Everything is working!

**Don't see messages?** = Read `GALLERY_TROUBLESHOOTING.md`
