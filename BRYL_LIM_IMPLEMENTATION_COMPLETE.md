## 🎨 Design System

### **Bryl Lim Color Palette**
```css
/* Dark Theme (Default) */
--color-bg-primary: #0a0a0a;        /* Deep black backgrounds */
--color-bg-secondary: #1a1a1a;      /* Section backgrounds */
--color-text-primary: #ffffff;       /* Main text */
--color-text-secondary: #b3b3b3;     /* Body text */
--color-accent: #4ade80;             /* Green accent for CTAs */

/* Light Theme */
--color-bg-primary: #ffffff;         /* White backgrounds */
--color-bg-secondary: #f8fafc;       /* Light gray sections */
--color-text-primary: #1e293b;       /* Dark text */
--color-accent: #3b82f6;             /* Blue accent */
```

### **Component Classes**
- `.card` - Main card styling with shadows and borders
- `.btn-primary` - Primary action buttons with hover effects
- `.btn-secondary` - Secondary buttons with outlined style
- `.badge` - Technology and achievement badges
- `.badge-sm` - Smaller badges for compact areas
