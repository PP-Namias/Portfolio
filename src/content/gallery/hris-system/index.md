## Project Overview

This enterprise-grade HRIS system was developed during my OJT internship at Aeternitas Chapels & Columbarium, where I led a team of 7 developers to create a comprehensive human resource management solution. The project addressed critical HR operational inefficiencies and provided a modern, secure platform for employee data management.

## Technical Architecture

The system was built using a modern tech stack with React and TypeScript for the frontend, packaged as an Electron desktop application for enhanced security and performance. The backend utilizes Express.js with PostgreSQL for robust data management and features comprehensive API documentation.

### Key Technical Decisions

- **Electron Desktop App**: Chosen for enhanced security and offline capabilities
- **PostgreSQL Database**: Selected for complex relational data handling and ACID compliance
- **Modular Architecture**: Designed for easy maintenance and feature expansion
- **Role-Based Access Control**: Implemented comprehensive permission system
