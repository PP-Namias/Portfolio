This project involves creating a car dealership management program entirely in C++. It was developed as part of the Fundamentals of Programming course.

- **Inventory Management**: Manage car inventory efficiently.
- **Sales Tracking**: Keep track of sales and customer data.
- **User-Friendly Interface**: Designed for ease of use.

## Technologies Used

- **C++**: For building the application.

## Outcome

The program has been successfully used as a learning tool for programming fundamentals, showcasing the power of C++ in building real-world applications.
