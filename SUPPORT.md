# Support

## Need Help?

Use one of the following channels depending on your need:

- **Bug report / feature request:** GitHub Issues
- **Security concern:** see `SECURITY.md` (private reporting)
- **General collaboration inquiry:** `pp.namias@gmail.com`

## Before Opening an Issue

Please include:

- Clear problem statement
- Steps to reproduce (if bug)
- Expected vs actual behavior
- Environment details (OS, Node version, browser)
- Screenshots/logs when relevant

## Local Troubleshooting Quick Checks

Run these commands before reporting runtime/build issues:

- `npm run lint`
- `npm run build`
- `npm run test`

## Project Scope Reminder

This repository is a personal portfolio project, but contributions and constructive feedback are welcome.
